### Compiler Version Requirements

| Compiler | Version         | Flags        |
| -------- | --------------- | ------------ |
| GCC      | >=10            | -fmodules-ts |
| Clang    | >=11            | -fmodules    |
| MSVC     | >=2019 Update 3 | -            |

### CMake Version Requirement

**CMake >= 3.28 (Support CXX_MODULE_SETS)**

| OS      | Tool  | Version         |
| ------- | ----- | --------------- |
| Linux   | Ninja | >=1.11          |
| Windows | MSVC  | >=2019 Update 3 |

### Supports File Extensions

- `.ixx`
